# How to run project

## Back End
Prerequisites: Java 11

```bash
# clone repository
git clone https://github.com/AugustoCesar/bds10-challenge

# enter folder backend
cd backend

# executar o projeto
./mvnw spring-boot:run
```

## Front End
Prerequisites: npm / yarn

```bash
# clone repository
git clone https://github.com/AugustoCesar/bds10-challenge

# enter folder frontweb
cd frontweb

# install dependencies
yarn install

# run project
yarn start

# run all tests
yarn test:watch

on menu press "a"
```
